Libevent
post time: 2017-07-29
网络

这个库太有名了, 相信我不介绍大家都知道了.

本文主要总结libevent相关的经验和乐趣.

不过有趣的是它自己的介绍:

$ sudo apt-cache search libevent
libevent-2.0-5 - Asynchronous event notification library
libevent-core-2.0-5 - Asynchronous event notification library (core)

然后你去它的官网, 它是说: 

Libevent includes support for several useful protocols, including DNS, HTTP, and a minimal RPC framework.

个人觉得他就是一个支持多路异步IO的轻量级网络框架, 其之后的libev对其事件做了进一步的封装.

引子

本文要求你有epoll, select等基础, 或者已经非常熟悉的情况下. 因为下面会剖析源码流程.

正文

安装

它支持autoconf, cmake等多种安装方式, 你怎么方便怎么来吧(没有cmake文件夹的源码, 只能用下面的方式).

## github的网络永远不怎么快
$ wget https://github.com/libevent/libevent/releases/download/release-2.1.8-stable/libevent-2.1.8-stable.tar.gz
$ tar -xzvf libevent-2.1.8-stable.tar.gz
$ cd libevent-2.1.8-stable/

$ ./configure 
$ make -j4
$ make verify   # (optional)
$ sudo make install
...
/bin/mkdir -p '/usr/local/include'
 /usr/bin/install -c -m 644 include/evdns.h include/event.h include/evhttp.h include/evrpc.h include/evutil.h '/usr/local/include'
 /bin/mkdir -p '/usr/local/include/event2'

查看是否安装成功(主要是看/usr/local/include和/usr/local/lib下是否有开发必备的头文件和库文件)

$ sudo ldconfig
$ ldconfig -p | grep libevent
  libevent_pthreads-2.1.so.6 (libc6,x86-64) => /usr/local/lib/libevent_pthreads-2.1.so.6
  libevent_extra-2.1.so.6 (libc6,x86-64) => /usr/local/lib/libevent_extra-2.1.so.6
  libevent_core-2.1.so.6 (libc6,x86-64) => /usr/local/lib/libevent_core-2.1.so.6
  libevent-2.1.so.6 (libc6,x86-64) => /usr/local/lib/libevent-2.1.so.6

$ ls /usr/local/include
evdns.h  event2  event.h  evhttp.h  evrpc.h  evutil.h

安装就说到这儿.
