using-regex-in-java

Java中已经有相当成熟的包提供了正则表达式支持, 直接引用就可以了.

oracle官方的java tutorials里面有相关章节专门讲这个主题,也很详细, 如下:
https://docs.oracle.com/javase/tutorial/essential/regex/

个人觉得太啰嗦了, 本文只写java.util.regex的主要(核心)内容.

引子

java.util.regex这个包主要涉及以下几个类

  1. Pattern
  2. Matcher
  3. PatternSyntaxException

主要用于2个方面:

  1. Match String
  2. Capture Group

注意java.util.regex属于标准库, 和PCRE也是兼容的.

正文

正文内容可以参考:
https://regexone.com/references/java

尾巴

上面已经演示了最常见的用法:

  1. Pattern.compile();
  2. pattern.matcher();
  3. matcher.replace相关

若有更多的需求, 可以详细参考一下Pattern Class和Matcher Class两个的相关源码和文档.
